Why This Platform?

  • ✅Combines discovery + personal curation.
  • ✅No external account or server required.
  • ✅Private offline storage stays in your browser.
  • ✅Installable like a native app (PWA).
  • ✅Resilience: works even when network is down.

How It Works

  • ⚙️IndexedDB stores files & metadata locally.
  • ⚙️Service Worker caches layout & assets.
  • ⚙️Search calls Google Books API (online mode).
  • ⚙️Auth uses localStorage with hashing (demo only).
  • ⚙️Export bundles base64-embed raw file blobs.

Online Reading vs Offline Library
